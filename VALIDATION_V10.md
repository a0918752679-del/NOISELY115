# v10 驗算紀錄

## 修正項目

首頁成案件數原先可能使用 `執行成效資料` 的彙總欄位加總，當 Google Sheet 匯入的彙總欄位重複或與車輛明細不同步時，會出現放大數值。

v10 已改為：

```text
成案件數 = 車輛案件資料中「告發」筆數 + 「通檢」筆數
```

## 已驗算資料

目前連動資料：

```text
車輛案件資料：80 筆
告發：55 筆
通檢：25 筆
成案件數：55 + 25 = 80
```
